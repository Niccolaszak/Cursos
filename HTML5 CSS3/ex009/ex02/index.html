Tags em HTML

As tags são elementos que compõem a estrutura de um documento HTML. Elas são formadas por um nome, que é delimitado por sinais de menor e maior (< e >), e podem conter atributos, que são informações adicionais sobre a tag.

As tags são divididas em dois tipos: tags de abertura e tags de fechamento. As tags de abertura são formadas pelo nome da tag e, opcionalmente, pelos atributos. As tags de fechamento são formadas pelo nome da tag precedido por uma barra (/).
